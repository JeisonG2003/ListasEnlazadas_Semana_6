# **Ejercicios de Listas Enlazadas Semana 6**

Repositorio del proyecto desarrollado para la asignatura Estructura de Datos (E) en la Universidad Estatal Amazónica, carrera de Ingeniería en Tecnologías de la Información.

## **Descripción**

Este proyecto contiene la solución a ejercicios prácticos relacionados con el uso de listas enlazadas simples, desarrollados en el lenguaje de programación C#.
Los ejercicios fueron seleccionados del documento “Ejercicios Propuestos de Listas Enlazadas – Miguel Domínguez” y corresponden a la Semana 6 de la asignatura.

El trabajo se enfoca en comprender la estructura de los nodos, el enlace entre ellos y la aplicación de operaciones básicas como el recorrido de la lista, el conteo de elementos y la inversión del orden de los datos.
El proyecto fue desarrollado y ejecutado en Visual Studio Code, mostrando los resultados directamente en consola.

## **Ejercicios desarrollados**

- Ejercicio 1: Función que calcula el número de elementos de una lista enlazada.
- Ejercicio 2: Inversión del orden de los elementos de una lista enlazada.

Cada ejercicio se encuentra organizado en su respectiva carpeta, separando la clase Nodo, la clase ListaEnlazada y el archivo Program.cs.

## **Objetivos**

- Aplicar el uso de listas enlazadas como estructuras de datos dinámicas en C#.
- Comprender el manejo de nodos y referencias dentro de una lista enlazada.
- Reforzar los conceptos básicos de Programación Orientada a Objetos.
- Desarrollar programas de consola claros, ordenados y funcionales.
- Organizar correctamente un proyecto en C# y publicarlo en un repositorio público de GitHub.
